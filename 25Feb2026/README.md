# Ejercicio de Maquetación CSS

Este proyecto es un ejercicio práctico para comprender el **Modelo de Caja (Box Model)** y la estructuración básica de elementos en una página web.

## 📝 Descripción
El objetivo de este ejercicio fue replicar un diseño específico aplicando estilos de CSS a elementos HTML. Se trabajó con propiedades fundamentales para controlar el espaciado y la apariencia de los componentes.

## 🛠 Tecnologías Utilizadas
* **HTML5**
* **CSS3**

## 📂 Estructura de Archivos
* `index.html`: Define la estructura jerárquica con las clases `caja1` y `caja2`.
* `ejercicioej.css`: Contiene las reglas de estilo que definen el color, tamaño y espaciado de los elementos.

## 💡 Aprendizajes Clave
* **Modelo de Caja (Box Model):**
    * `padding`: Espacio interno entre el contenido y el borde.
    * `margin`: Espacio externo que separa las cajas entre sí.
    * `border`: Definición de grosor y estilo de los bordes.
* **Centrado de contenedores:** Uso de `margin: 0 auto;` para centrar el `body` en la pantalla.
* **Selectores:** Aplicación eficiente de estilos mediante clases (`.caja1`, `.caja2`).

## 🚀 Cómo visualizar el proyecto
1. Asegúrate de tener los archivos `index.html` y `ejercicioej.css` en la misma carpeta.
2. Abre el archivo `index.html` en cualquier navegador web moderno.
